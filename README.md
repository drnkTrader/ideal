OPT6 ise İzleyen stopun ma2'ye olan uzaklığı gene binde X cinsinden

Bu sistemi kesinlikle canlı kullanım için değil, 
dinamik bir TP ve STOP yapısı nasıl kurgulanabilir göstermek için paylaşıyorum.
